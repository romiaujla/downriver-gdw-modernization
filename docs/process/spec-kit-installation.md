# Spec Kit Installation

This repository uses Spec Kit as the planning workflow for `--speckit GRA-<id>` delivery.

## Baseline Expectation

- Spec Kit commands should be available in the local project environment before starting a `--speckit` flow.
- Planning artifacts should live under `specs/GRA-<id>/`.

## Required Artifacts

- `specs/GRA-<id>/spec.md`
- `specs/GRA-<id>/plan.md`
- `specs/GRA-<id>/tasks.md`
- `specs/GRA-<id>/wireframe.md` for UI work

## Setup Note

- If Spec Kit is not yet installed in this repository, install it before starting a new `--speckit` workflow so the tagged command contract in `AGENTS.md` remains actionable.
